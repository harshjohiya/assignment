# Assignment Project

This project contains notebook-based experiments for several generative models, including VAE, GAN, Transformer, and Diffusion.

## Contents

- `notebooks/` - experiment notebooks
- `model/` - saved model weights
- `evaluation/` - training curves and evaluation visuals
- `results/` - generated sample images

## Notes

- Open the notebooks to explore the experiments.
- The saved models and result images are included for reference.
